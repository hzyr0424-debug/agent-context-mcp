# agent-context-mcp

A Model Context Protocol server for managing agent context, workflow memory, and task state.

## What is this?

`agent-context-mcp` is an MCP server designed to help AI agents store, retrieve, and manage context across tasks.

It focuses on:

- Agent context management
- Workflow memory
- Task state tracking
- Long-running agent collaboration
- Developer-friendly MCP integration

## Use cases

- Keep agent memory across multiple tool calls
- Store project-level context for coding agents
- Track task progress in multi-agent workflows
- Provide structured context to MCP-compatible clients
